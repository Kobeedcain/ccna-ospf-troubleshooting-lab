# ccna-ospf-troubleshooting-lab
CCNA OSPF troubleshooting lab using Cisco Packet Tracer demonstrating OSPF configuration, neighbor verification, and route propagation.

## Overview
This project demonstrates troubleshooting and verification of OSPF routing in a multi-router topology using Cisco Packet Tracer.

## Objectives
- Restore OSPF neighbor adjacencies
- Troubleshoot route advertisement issues
- Configure OSPF on newly added network links
- Verify connectivity to external networks
- Analyze the OSPF Link-State Database (LSDB)

## What I Learned
- Configured OSPF in a multi-router Area 0 topology
- Restored OSPF neighbor adjacencies between routers
- Troubleshot route propagation issues
- Configured an ASBR to advertise external routes
- Verified routing tables using Cisco IOS commands
